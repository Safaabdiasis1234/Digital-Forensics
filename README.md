# Digital Forensics — CMP020N210S

BSc Cyber Security | University of Roehampton

This repository contains two forensic investigation courseworks completed as part of the Digital Forensics module. Both investigations were conducted using industry-standard tools in line with ACPO best practice guidelines.

---

## Portfolio 1 — Analysing Forensic Images

Investigation of a suspected piracy and criminal activity case. Two forensic USB images were examined to recover hidden, deleted, and encrypted files. Key findings included a VeraCrypt encrypted container, suspicious files in the Recycle Bin, and individuals linked to illegal activity through metadata and file analysis.

**Tools:** Autopsy, FTK Imager

---

## Portfolio 3 — Forensic Analysis of an Attack

Investigation of a suspected insider attack on a Windows 10 virtual machine. Registry hives, BAM logs, prefetch files, and file system metadata were analysed to reconstruct a timeline of malicious activity on 22nd March 2024. Key findings included execution of a PowerShell attack script (ART-attack.ps1), creation of a rogue admin account, and use of the AdFind enumeration tool.

**Tools:** Autopsy, Registry Explorer, Volatility
